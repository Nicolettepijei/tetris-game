# reductris

A puzzle game that is just for funsies. Inspired by [Meth Meth Method's video](https://www.youtube.com/watch?v=H2aW5V46khA).

Uses:

- React
- Redux
- Ducks module organization
- AVA for unit tests

## Controls

| Key   | Action |
| ----- | ------ |
| Left, A  | Left   |
| Right, D | Right  |
| Down, S  | Down   |
| Shift, Up    | Rotate Right |
| Z, /    | Rotate Left |
| Enter    | Pause / Restart Game |
